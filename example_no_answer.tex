\section{示例题目}

\subsection{选择题示例}

\begin{qitems}
\qitem 下列关于数据结构的说法正确的是：

\fourchoices{数据结构只包含数据}{数据结构包含数据和操作}{数据结构就是算法}{数据结构不重要}

\answer{B. 数据结构包含数据和对数据的操作}

\qitem 栈的特点是：

\fourchoices{先进先出}{后进先出}{随机存取}{顺序存取}

\answerbox{B. 后进先出（LIFO - Last In First Out）是栈的基本特征}
\end{qitems}

\subsection{填空题示例}

3. 二叉树的遍历方式有\_\_\_\_、\_\_\_\_、\_\_\_\_三种。

\answer{前序遍历、中序遍历、后序遍历}

4. 完全二叉树具有n个节点，则该树的高度为\_\_\_\_。

\answerbox{$\lfloor \log_2 n \rfloor + 1$}
